# Wordpress-Ecommerce
Wordpress Ecommerce without plans
Tienda online creada como proyecto académico. Incluye un catálogo de productos, carrito de compra y proceso de pago, construido íntegramente con herramientas gratuitas y de código abierto.

⚠️-------
Este proyecto utiliza únicamente plugins y temas gratuitos.
No se ha adquirido ninguna licencia de pago ni se incluye software premium en el repositorio.

**Desarrollado con fines educativos como parte de un trabajo de clase. No está orientado a producción ni uso comercial.**
